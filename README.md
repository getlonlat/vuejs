# getLonLat vue.js

## Project setup
```
yarn install
```

## Env setup
Create a .env and add a Google Maps api key with Maps API
and Geocoding API permissions.
```
cp .env.example .env.local
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
